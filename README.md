## ✨ Features

- 👨‍🌾 **Farmer** — Product listing, order management, revenue analytics, weather advisory
- 🛒 **Consumer** — Shop, cart, UPI payment, delivery tracking, recommendations
- 🤖 **AI Disease Detector** — Crop disease diagnosis with treatment plans
- ⚙️ **Admin** — User management, product moderation, activity logs
- 🌍 **Multi-language** — English, हिन्दी, தமிழ், తెలుగు
- 🔥 **Firebase** — Real-time sync across devices
- 📲 **PWA** — Installable on mobile, works offline

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/agrismart.git
# Open index.html in any browser — no setup needed
```

For Firebase sync → open app → **⚙️ Firebase Configuration** → paste your config.  
For demo mode → click **💾 Use Local Storage**.
